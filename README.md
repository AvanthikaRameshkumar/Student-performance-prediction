# Student-performance-prediction

## Description :
 This project predicts student performance based on factors like study hours and attendance using Python and SQLite database. It helps analyze how different parameters affect student marks.
## Features :
 Stores student data using SQLite database  - Analyzes study hours and attendance  - Predicts student marks  - Simple and efficient data handling.
## Technologies Used :
 Python  - SQLite  - Google Colab.
